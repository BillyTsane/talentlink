# 🚀 TalentLink

TalentLink is a platform connecting sports talents with clubs, agents and recruiters through video performance and smart discovery.

## 🧱 Tech Stack

- Backend: Django + DRF
- Database: PostgreSQL
- Cache: Redis
- Storage: AWS S3
- Containerization: Docker
- CI/CD: GitHub Actions

## ⚙️ Local Setup

docker-compose up --build
